Official Site Plan Document

Site Name

Site Name: TableTop Command

This name reflects the RTS-inspired mechanics of the board game, combining the idea of large-scale strategic conquest with physical board game counters and units. It signals tactical depth, battlefield command, and strategic decision-making.

Optional Domain: tabletopcommand.com

Site Purpose

The purpose of this site is to introduce players to the RTS-inspired board game TableTop Command. The site will provide:

  • Landing page for the site
  • Rulebook access and setup guides
  • Units showcase page.
  • Building showcase page

Scenarios

These are questions a target player might have when visiting the site:

  • How does this game feel like an RTS if it’s a board game?
  • What factions can I play and how are they different?
  • Where can I download or read the rulebook?
  • Is this game better for competitive or casual play?

Color Schema

Primary Color – Deep Strategy Blue (#1f2a44)

  • Used for headers, footer, and major headings
  • Represents strategy, control, and focus

Secondary Color – Tactical Gold (#c2a83e)

  • Used for accents, borders, and highlights
  • Represents victory, value, and elite units

Light Neutral (#f4f6f8)

  • Main page background
  • Keeps content readable and clean

Typography

Orbitron (Sans-serif)

  • Used for headings and titles
  • Gives a futuristic, tactical, RTS-style feel

Roboto (Sans-serif)

  • Used for body text and paragraphs
  • Clean and highly readable for instructions and rules
